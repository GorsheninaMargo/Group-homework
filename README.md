# Групповое домашнее задание
## Выполнение
### Программа работает по следующему алгоритму:
```
1. Пользователю предлагается выбор метода сортировки, после выбора которого пользователь 
должен задать множество чисел, которые необходимо отсортировать
2. Пользователю предлагается выбор создания массива: случайно или ввод элементов вручную.
При ручном вводе пользователь вводит числа, пока не введет в консоль символ не являющийся цифрой.
При случайном задании массива чисел пользователю предлагается выбрать диапазон значений и количество 
элементов массива.
3. Далее множество сортируется, согласно выбранному методу.
```
---
> Метод выбором:
```
1)	Находим номер минимального значения в текущем списке

2)	Производим обмен этого значения со значением первой не
отсортированной позиции (обмен не нужен, если минимальный элемент уже находится на данной позиции)

3)	Теперь сортируем конец списка, исключив из рассмотрения уже отсортированные элементы

```
---
> Метод слиянием:
```
Задача разбивается на подзадачи, затем их решения комбинируются для получения решения исходной задачи. 
Конкретно процедуру сортировки слиянием можно описать следующим образом:

1)	Если рассматриваемый массив состоит из одного элемента, то он уже отсортирован — алгоритм завершает работу.
 
2)	Иначе массив разбивается на две части, которые сортируются рекурсивно.

3)	После сортировки двух частей массива к ним применяется процедура слияния, которая по двум отсортированным частям получает исходный
отсортированный массив.

```
